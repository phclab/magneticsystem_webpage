Open-Source Magnetic System for Wireless Neuromodulations

Published in Scientific Reports (2025)
Abstract

This repository provides the open-source hardware designs and software controlled for the magnetic system described in our research. The system is designed to enable high-precision wireless neuromodulation, supporting both in vitro applications and in vivo untethered brain stimulation. By providing these resources, we aim to facilitate the adoption and further development of magnetic stimulation technologies in the neuroscience community.

Resources & Downloads

All source files, including CAD models for hardware components and Python-based control scripts, are available on our GitHub repository.
