Magnetic System

Python based, open-source electromagnetic coil control system.

Overview

This Magnetic System is a open-source system which can be used to control electromagnetic coils. It is consisting of a console board and graphical user interface.

This System can establish customize experiment protocol, and control electromagnetic coils with experiment protocol setting. With sensors or webcam, This System ensures the experiment situation is stable.

The hardware, software and documentation repositories are hosted on GitHub, along with a console board for user support and questions.

The research for Magnetic System lists a feasible experiment that you can follow in your own experiment. You can also establish your own experiment with Our System.

Citation

Getting started

Dependencies

Magnetic System has the following dependencies:

  • Python 3.8.8
  • pyserial 3.5
  • PyQt 5.15.7
  • You can download the environment.yml with the button and use the code below to establish the system environment.

On a windows machine that does not already have Python installed, the simplest way to install Python is to download an installer from python.org. To ensure you can run Python files (including the Magnetic System GUI) by double clicking on them, select the options 'Add Python to PATH', 'Add Python to environment variables', and 'Associate files with Python' during installation. Note; selecting these options when installing Python will cause the Python version you are installing to take precedence over any other version of Python installed on the computer, so may break previous Python based workflows on the computer.

Once you have a working Python 3.8.8 installation on the computer, run the windows command prompt (cmd.exe) as an administrator and enter the following commands to install the required packages:

python -m pip install pyserial 3.5
            python -m pip install pyqt 5.15.7

Magnetic System has been tested primarily on Windows 10 but in principle should be cross platform.

Installation

Get the latest version of Magnetic System, either by downloading it at a zip file from the releases page or by cloning the code repository to your computer with Git. Cloning the repository is recommended as it makes it easier to update Magnetic System in the future. Magnetic System is in active development with new versions released from time to time. The default documentation is for the latest release, but the dropdown menu in the bottom right corner of the docs lets you access documentation for earlier versions, and for the dev branch of the code repository where new features are developed.

Running an experiment

Load the GUI by running the file magnetoEcosystem.pyw.

Connect a console board to the computer with a USB cable. The GUI will get the board automatically.

Select the Protocol tab. Then, you can select the sensors or cameras used in the experiment. You can set the threshold to make the system stop immediately while the real-time sensor value is over.

You can also choose the stimulation pattern and set the stimulation protocol with the table.

Select the Camera tab, you can select the type of the mice behavior tests, the color of the background, and the method to control the electromagnetic coils in the dropdowns.

You can set the regions of interest with the Region button. After pressing the Region button, you should first take a screenshot by pressing the left button of the mouse. Then, you should select four points to form the region of interest in the Light-dark box test and eight points in the Place Preference test.

After selecting the regions of interest, you can use the threshold as a standard to check whether the mice are in the area.

Select the Experiment tab and press Start to start stimulation.

If you encounter problems take a look at the troubleshooting page.
