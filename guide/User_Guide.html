User Guide

The MagneticSystem contains graphic user interface and hardware.

Graphical user interface (GUI)

The whole system consists of 4 pages, including "Protocol Page", "Sensor Page", "Camera Page" and "Stimulation Page".

Protocol Page

【ToolBar】

"New File" button is used for creating the blank protocol.

"Save File" button is used for saving the current protocol.

"Load File" button is used for loading the previous protocol.

"Add a new Column" button is used for adding the column after the last column in the protocol.

"Remove Last Column" button is used for removing the last column in the protocol.

"Search Arduino" button is used for connecting to the Arduino.

【Stimulation Mode】

"On-Off" means generate the alternative magnetic field in one direction.

"Biderection" means generate the alternative magnetic field in both direction.

"Always On" means generate the static magnetic field.

【Protocol Table】

"Action" row is used for opening or closing the magnetic field in the period.

"Duration(sec)" row is used for setting the durarion of the period.

"Frequency(Hz)" row is used for setting the frequency of the alternative magnetic field in the period.

"Dead Time(millisecond)" row is used for setting the dead time between each cycle in the period.

【Protocol Plot】

The plot shows the magnetic field operation with the protocol.

Sensor Page

【Control Mode】

Set whether the magnetic field be stopped while the sensor value is higher than the threshold.

【Sensor Detection】

Set the sensors opened in the stimulation and the threshold of each sensor.

Camera Page

【ToolBar】

"Play Video" button is used for starting the detection in the video or the stream.

"Load Video File" button is used for loading the video.

"Clear Detection ROI" button is used for resetting the area of interest.

"Stop All the System" button is used for stopping the stimulation.

【Detect Setting】

Set the mice behavior test, background color, magnetic field stopped mode and the threshold of the detection.

【Realtime Information】

The plots Show the magnetic field operation with the protocol, during the real-time stimulation and the position of the mice.

The photo Shows the current frame of the video or the stream.

Stimulation Page

【Buttons】

"Start" button is used for starting the stimulaion.

"Stop" button is used for stopping the stimulation.

【Loading Bar】

Shows the progression of the stimulation.

【Signal Plot】

The plots Show the magnetic field operation with the protocol and during the real-time stimulation.

【Sensor Plot】

The plots Show the current value from each sensor.

Hardware

The console board with an Arduino Nano can control a maximum of 6 electromagnets.

The Arduino Nano generates a 5V signal that fed into NOT (SN74HC04N, Texas Instruments) and AND (SN74HC08N, Texas Instruments) logic gate IC.

On the console board, 3 AND logic chips(SN74HC08N, Texas Instruments), 2 NOT logic chips(SN74HC04N, Texas Instruments), and 6 op-amp chips (TLC2272) were used.

The Arduino Nano contains 12 pins for digital output, D2 to D13, and 7 pins for analog input/output, A1 to A7.

Pin A1 and pin A2 were used to output the signal to external devices. A3 was used for the Hall sensor reading. A4 and A5 were used for the acceleration sensor reading. A6 was used for temperature sensor reading. A7 was used for decibel sensor reading.
